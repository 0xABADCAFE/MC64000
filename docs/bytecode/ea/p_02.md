## General Purpose Register Indirect

The contents of the register are used as the address of the operand data in memory.

General syntax: `(r<N>)`

* Allowed register names: r0 ... r15, a0 ... a7, sp
* All bits of the register are used.

| Mode | Bytecode |
| - | - |
| `(r0)` | 0x10 |
| `(r1)` | 0x11 |
| ... | ... |
| `(r15)` | 0x1F |
