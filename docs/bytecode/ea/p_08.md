## Floating Point Register Direct

The contents of the register are used as the operand data. This mode is only available for floating point operations.

General syntax: `fp<N>`

* Allowed register names: fp0 ... fp15

| Mode | Bytecode |
| - | - |
| `fp0` | 0x70 |
| `fp1` | 0x71 |
| ... | ... |
| `fp15` | 0x7F |
