## FTWOTOX

Floating Point 2 to x

        2.0^<ea(s)> -> <ea(d)>

General syntax:

        ftwotox.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `ftwotox.s <ea(s)>, <ea(d)>` | 0xEA | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `ftwotox.d <ea(s)>, <ea(d)>` | 0xEB | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
