## OR

Bitwise OR.

        <ea(s)> | <ea(d)> -> <ea(d)>

General syntax:

        or.<b|w|l|q> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `or.b <ea(s)>, <ea(d)>` | 0x76 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `or.w <ea(s)>, <ea(d)>` | 0x77 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `or.l <ea(s)>, <ea(d)>` | 0x78 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `or.q <ea(s)>, <ea(d)>` | 0x79 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
