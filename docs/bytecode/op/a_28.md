## FLOGN

Floating Point Natural Logarithm.

        ln(<ea(s)>) -> <ea(d)>

General syntax:

        flogn.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `flogn.s <ea(s)>, <ea(d)>` | 0xE6 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `flogn.d <ea(s)>, <ea(d)>` | 0xE7 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
