## FLOG2

Floating Point Base-2 Logarithm.

        log2(<ea(s)>) -> <ea(d)>

General syntax:

        flog2.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `flog2.s <ea(s)>, <ea(d)>` | 0xE8 | *EA*(d) | ... | *EA*(s) | ... |
| `flog2.d <ea(s)>, <ea(d)>` | 0xE9 | *EA*(d) | ... | *EA*(s) | ... |
