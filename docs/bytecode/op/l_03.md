## EOR

Bitwise Exclusive OR.

        <ea(s)> ^ <ea(d)> -> <ea(d)>

`eor.<b|w|l|q> <ea(s)>, <ea(d)>`

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `eor.b <ea(s)>, <ea(d)>` | 0x7A | *EA*(d) | ... | *EA*(s) | ... |
| `eor.w <ea(s)>, <ea(d)>` | 0x7B | *EA*(d) | ... | *EA*(s) | ... |
| `eor.l <ea(s)>, <ea(d)>` | 0x7C | *EA*(d) | ... | *EA*(s) | ... |
| `eor.q <ea(s)>, <ea(d)>` | 0x7D | *EA*(d) | ... | *EA*(s) | ... |
