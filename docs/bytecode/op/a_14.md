## DIVU

Integer Unsigned Divide.

        <ea(d)> / <ea(s)> -> <ea(d)>

General syntax:

        divu.<l|q> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `divu.l <ea(s)>, <ea(d)>` | 0xCA | *EA*(d) | ... | *EA*(s) | ... |
| `divu.q <ea(s)>, <ea(d)>` | 0xCB | *EA*(d) | ... | *EA*(s) | ... |
