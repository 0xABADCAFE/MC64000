## CLR

Clear a location.

        0 -> <ea>

General syntax:
        clr.<b|w|l|q> <ea>

* All bits are set to zero.

| Mnemonic | Bytecode | Ext 0 | ... |
| - | - | - | - |
| `clr.b <ea>`| 0x65 | *EA* | ... |
| `clr.w <ea>`| 0x66 | *EA* | ... |
| `clr.l <ea>`| 0x67 | *EA* | ... |
| `clr.q <ea>`| 0x68 | *EA* | ... |
