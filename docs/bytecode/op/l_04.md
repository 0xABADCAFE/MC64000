## NOT

Bitwise Complement.

        ~<ea(s)> -> <ea(d)>

General syntax:

        not.<b|w|l|q> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `not.b <ea(s)>, <ea(d)>` | 0x7E | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `not.w <ea(s)>, <ea(d)>` | 0x7F | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `not.l <ea(s)>, <ea(d)>` | 0x80 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `not.q <ea(s)>, <ea(d)>` | 0x81 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
