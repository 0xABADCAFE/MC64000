## EXTL

Sign Extend Long.

        <ea(s)> -> <ea(d)>

General syntax:

        extl.q <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `extl.q <ea(s)>, <ea(d)>` | 0xA1 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
