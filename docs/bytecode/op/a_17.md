## FMOD

Floating Point Modulus.

        <ea(d)> % <ea(s)> -> <ea(d)>

General syntax:

        fmod.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `fmod.s <ea(s)>, <ea(d)>` | 0xD0 | *EA*(d) | ... | *EA*(s) | ... |
| `fmod.d <ea(s)>, <ea(d)>` | 0xD1 | *EA*(d) | ... | *EA*(s) | ... |
