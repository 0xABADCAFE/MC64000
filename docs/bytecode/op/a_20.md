## FACOS

Floating Point Arccosine.

        acos(<ea(s)>) -> <ea(d)>

General syntax:

        facos.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `facos.s <ea(s)>, <ea(d)>` | 0xD6 | *EA*(d) | ... | *EA*(s) | ... |
| `facos.d <ea(s)>, <ea(d)>` | 0xD7 | *EA*(d) | ... | *EA*(s) | ... |
