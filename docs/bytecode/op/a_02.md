## EXTW

Sign Extend Word.

        <ea(s)> -> <ea(d)>

General syntax:

        extw.<l|q> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `extw.l <ea(s)>, <ea(d)>` | 0x9F | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `extw.q <ea(s)>, <ea(d)>` | 0xA0 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
