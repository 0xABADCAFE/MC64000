## EXTW

Sign Extend Word.

        <ea(s)> -> <ea(d)>

General syntax:

        extw.<l|q> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `extw.l <ea(s)>, <ea(d)>` | 0x9F | *EA*(d) | ... | *EA*(s) | ... |
| `extw.q <ea(s)>, <ea(d)>` | 0xA0 | *EA*(d) | ... | *EA*(s) | ... |

### Fast Path Variants

| Mnemonic | Prefix | Reg | Opcode |
| - | - | - | - |
| `extw.l r<S>, r<D>` | 0xFE | S:D | 0x9F |
| `extw.q r<S>, r<D>` | 0xFE | S:D | 0xA0 |
