## JMP

Jump.

        <ea> -> pc

General syntax:

        jmp <ea>

* Effective Address cannot be register direct or integer immediate.

| Mnemonic | Bytecode | Ext 0 | ... |
| - | - | - | - |
| `jmp <ea>`| 0x05 | 0x*EA* | ... |
