## FMUL

Floating Point Multiply.

        <ea(d)> * <ea(s)> -> <ea(d)>

General syntax:

        fmul.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `fmul.s <ea(s)>, <ea(d)>` | 0xC4 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `fmul.d <ea(s)>, <ea(d)>` | 0xC5 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
