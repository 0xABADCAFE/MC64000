## AND

Bitwise AND.

        <ea(s)> & <ea(d)> -> <ea(d)>

General syntax:

        and.<b|w|l|q> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `and.b <ea(s)>, <ea(d)>` | 0x72 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `and.w <ea(s)>, <ea(d)>` | 0x73 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `and.l <ea(s)>, <ea(d)>` | 0x74 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `and.q <ea(s)>, <ea(d)>` | 0x75 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
