## FDIV

Floating Point Divide.

        <ea(d)> / <ea(s)> -> <ea(d)>

General syntax:

        fdiv.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `fdiv.s <ea(s)>, <ea(d)>` | 0xCE | *EA*(d) | ... | *EA*(s) | ... |
| `fdiv.d <ea(s)>, <ea(d)>` | 0xCF | *EA*(d) | ... | *EA*(s) | ... |

### Fast Path Variants

| Mnemonic | Prefix | Reg | Opcode |
| - | - | - | - |
| `fdiv.s r<S>, r<D>` | 0xFF | S:D | 0xCE |
| `fdiv.d r<S>, r<D>` | 0xFF | S:D | 0xCF |
