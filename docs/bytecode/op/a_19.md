## FSQRT

Floating Point Square Root.

        sqrt(<ea(s)>) -> <ea(d)>

General syntax:

        fqrt.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `fsqrt.s <ea(s)>, <ea(d)>` | 0xD4 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `fsqrt.d <ea(s)>, <ea(d)>` | 0xD5 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
