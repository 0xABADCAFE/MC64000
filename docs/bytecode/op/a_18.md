## FABS

Floating Point Absolute.

        abs(<ea(s)>) -> <ea(d)>

General syntax:

        fabs.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `fabs.s <ea(s)>, <ea(d)>` | 0xD2 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `fabs.d <ea(s)>, <ea(d)>` | 0xD3 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
