## FETOX

Floating Point e to x.

        e^<ea(s)> -> <ea(d)>

General syntax:

        fetox.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `fetox.s <ea(s)>, <ea(d)>` | 0xE4 | *EA*(d) | ... | *EA*(s) | ... |
| `fetox.d <ea(s)>, <ea(d)>` | 0xE5 | *EA*(d) | ... | *EA*(s) | ... |
