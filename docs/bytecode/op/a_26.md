## FTAN

Floating Point Tangent.

        tan(<ea(s)>) -> <ea(d)>

General syntax:

        ftan.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `ftan.s <ea(s)>, <ea(d)>` | 0xE2 | *EA*(d) | ... | *EA*(s) | ... |
| `ftan.d <ea(s)>, <ea(d)>` | 0xE3 | *EA*(d) | ... | *EA*(s) | ... |
