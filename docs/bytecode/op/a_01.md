## EXTB

Sign Extend Byte.

        <ea(s)> -> <ea(d)>

General syntax:

        extb.<w|l|q> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `extb.w <ea(s)>, <ea(d)>` | 0x9C | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `extb.l <ea(s)>, <ea(d)>` | 0x9D | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `extb.q <ea(s)>, <ea(d)>` | 0x9E | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
