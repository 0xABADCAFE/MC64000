## BFCNT

Count number of set bits.

        pop_count(<ea(s)>) -> <ea(d)>

General syntax:

        bfcnt <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `bfcnt <ea(s)>, <ea(d)>` | 0x9B | *EA*(d) | ... | *EA*(s) | ... |
