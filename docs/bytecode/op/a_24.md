## FSIN

Floating Point Sine.

        sin(<ea(s)>) -> <ea(d)>

General syntax:

        fsin.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `fsin.s <ea(s)>, <ea(d)>` | 0xDE | *EA*(d) | ... | *EA*(s) | ... |
| `fsin.d <ea(s)>, <ea(d)>` | 0xDF | *EA*(d) | ... | *EA*(s) | ... |
