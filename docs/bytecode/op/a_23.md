## FCOS

Floating Point Cosine.

        cos(<ea(s)>) -> <ea(d)>

General syntax:

        fcos.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `fcos.s <ea(s)>, <ea(d)>` | 0xDC | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `fcos.d <ea(s)>, <ea(d)>` | 0xDD | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
