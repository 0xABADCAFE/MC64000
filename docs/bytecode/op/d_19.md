## PEA

Push Effective Address.

        sp - 8 -> sp; <ea> -> (sp)

General syntax:

        pea <ea>

| Mnemonic | Opcode | Ext 0 | ... |
| - | - | - | - |
| `pea <ea>`| PEA | *EA* | ...|
