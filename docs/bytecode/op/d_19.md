## PEA

Push Effective Address.

        sp - 8 -> sp; <ea> -> (sp)

General syntax:

        pea <ea>

| Mnemonic | Bytecode | Ext 0 | ... |
| - | - | - | - |
| `pea <ea>`| 0x71 | 0x*EA* | ...|
