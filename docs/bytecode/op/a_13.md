## MODS

Integer Signed Modulus.

        <ea(d)> % <ea(s)> -> <ea(d)>

General syntax:

        mods.<l|q> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `divs.l <ea(s)>, <ea(d)>` | 0xC8 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `divs.q <ea(s)>, <ea(d)>` | 0xC9 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
