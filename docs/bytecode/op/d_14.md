## FEXG

Exchange FPR.

        fp<S> <-> fp<D>

`fexg fp<S>, fp<D>`

* Floating Point Register contents are exchanged.

| Mnemonic | Bytecode | Ext 0 |
| - | - | - |
| `fexg fp<S>, fp<D>`| 0x6A | S:D |
