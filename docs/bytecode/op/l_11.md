## BFFFO

Find First One in Bitfield.

General syntax:

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `<bfffo>` | 0x9A | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
