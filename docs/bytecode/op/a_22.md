## FATAN

Floating Point Arctangent.

        atan(<ea(s)>) -> <ea(d)>

General syntax:

        fatan.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `fatan.s <ea(s)>, <ea(d)>` | 0xDA | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `fatan.d <ea(s)>, <ea(d)>` | 0xDB | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
