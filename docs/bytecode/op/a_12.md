## DIVS

Integer Signed Divide.

        <ea(d)> / <ea(s)> -> <ea(d)>

General syntax:

        divs.<l|q> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `divs.l <ea(s)>, <ea(d)>` | 0xC6 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `divs.q <ea(s)>, <ea(d)>` | 0xC7 | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
