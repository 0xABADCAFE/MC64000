## FASIN

Floating Point Arcsine.

        asin(<ea(s)>) -> <ea(d)>

General syntax:

        fasin.<s|d> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `fasin.s <ea(s)>, <ea(d)>` | 0xD8 | *EA*(d) | ... | *EA*(s) | ... |
| `fasin.d <ea(s)>, <ea(d)>` | 0xD9 | *EA*(d) | ... | *EA*(s) | ... |
