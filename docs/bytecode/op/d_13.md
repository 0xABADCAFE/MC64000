## EXG

Exchange GPR.

        r<S> <-> r<D>

General syntax:

        exg r<S>, r<D>

* General Purpose Register contents are exchanged.

| Mnemonic | Bytecode | Ext 0 |
| - | - | - |
| `exg <S>, r<D>`| 0x69 | 0x*SD* |
