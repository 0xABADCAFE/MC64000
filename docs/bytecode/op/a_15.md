## MODU

Integer Unsigned Modulus.

        <ea(d)> % <ea(s)> -> <ea(d)>

General syntax:

        modu.<l|q> <ea(s)>, <ea(d)>

| Mnemonic | Bytecode | Ext 0 | ... | ... | ... |
| - | - | - | - | - | - |
| `modu.l <ea(s)>, <ea(d)>` | 0xCC | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
| `modu.q <ea(s)>, <ea(d)>` | 0xCD | 0x*EA*(d) | ... | 0x*EA*(s) | ... |
